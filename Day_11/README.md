### 🧩 Section 19 – Advanced SQL: CASE, Variables, Indexes & Triggers

Topics Covered:

- CASE Statement:  
  Conditional logic in SQL queries to categorize or transform data.

- Variables:  
  - Local Variables: Declared inside procedures/functions; scope limited to that block.  
  - Global Variables: Accessible across sessions or procedures.  
  - Session Variables: Live during a session; independent of other sessions.

- Indexes:  
  Improve query performance by creating indexes on columns frequently used in WHERE clauses.

- Triggers:  
  Automate actions on INSERT, UPDATE, or DELETE events in tables.

Key Takeaways:
- CASE Statements: Enable conditional logic directly in queries for dynamic categorization.  
- Variables: Local, global, and session variables allow flexible storage and computation within SQL queries, procedures, and sessions.  
- Indexes: Improve query performance by speeding up data retrieval on frequently accessed columns.  
- Triggers: Automate repetitive or rule-based actions on INSERT, UPDATE, or DELETE events, ensuring data integrity and consistency.
