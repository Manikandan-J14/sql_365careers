### ✍️ Day 6 – Sections 10 & 11: INSERT, UPDATE, COMMIT & ROLLBACK

Topics Covered:

Section 10 – INSERT:
- Basic syntax: INSERT INTO table (columns) VALUES (values);
- Inserting multiple rows at once
- Inserting values into specific columns
- Handling default values and nulls

Section 11 – UPDATE + Transaction Control:
- Updating existing records using UPDATE ... SET ... WHERE ...
- Avoiding dangerous updates by always using WHERE
- Use of expressions in SET
- Introduction to transactions:
  - COMMIT
  - ROLLBACK

Key Takeaways:
- Learned how to add and update data within existing tables.
- Understood importance of precision with WHERE to avoid accidental full-table updates.
- Practiced transaction control using COMMIT and ROLLBACK to ensure data safety.
- These are critical skills for any backend, database, or data engineer role.
